## 컨벤션

### 브랜칭 전략

- main 브랜치에는 푸쉬하지 않는다
  - 브랜치를 판 다음에, main으로 PR 걸기
  - PR 리뷰를 1개 이상 받은 후 머지
- 브랜치 명
  - `feat/~~~`: 기능 추가 (`feat/initialize-nextjs`)
  - `fix/~~~`
  - `refact/~~~`
  - `hotfix/~~~`
- PR 시에는 템플릿 따라 작성
